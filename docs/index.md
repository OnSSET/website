# OnSSET

**OnSSET (OpeN Source Spatial Electrification Tool)** is an open-source toolkit for geospatial electrification planning.

## Quick links
- Code: https://github.com/orgs/OnSSET/repositories
- Forum: https://forum.u4ria.org/t/potential-topics-for-new-developments/530
- Learning: https://www.open.edu/openlearncreate/course/view.php?id=16177
- Sample data (GEP): https://energydata.info/group/gep
- Global Electrification Platform: https://electrifynow.energydata.info/
