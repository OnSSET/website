# OnSSET

**OnSSET (OpeN Source Spatial Electrification Tool)** is an open-source toolkit for geospatial electrification planning.

## Start here
- **Run the tool:** https://github.com/OnSSET/onsset
- **Explore sample results & data (58 countries):** https://electrifynow.energydata.info/
- **Download sample datasets (GEP):** https://energydata.info/group/gep
- **Learn with guided material:** https://www.open.edu/openlearncreate/course/view.php?id=16177
