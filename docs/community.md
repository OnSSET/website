# Community

OnSSET is developed and maintained by an open international community of researchers, analysts, and policy practitioners. We welcome questions, discussions, feedback, and contributions.

---

## Where to Ask What

To keep discussions efficient and searchable, please use the appropriate channel:

### Modelling Questions & Methodological Discussions
For conceptual questions, interpretation of results, and modelling approaches, please use the [OnSSET Modelling Forum](https://forum.u4ria.org/t/potential-topics-for-new-developments/530). This is the preferred space for technical and methodological exchange.

---

### Bug Reports & Technical Issues
If something is not working as expected you can open an issue in the relevant repository under the  
  [OnSSET GitHub organisation](https://github.com/orgs/OnSSET/repositories)

Please include:

- Model version  
- Input description  
- Error message or behaviour observed  
- Steps to reproduce  

---

### Feature Requests & Development Proposals
For proposing new capabilities or improvements:

- Start a discussion in GitHub  
- Raise the topic in the modelling forum  
- Contribute through Steering Committee consultations  

Clear problem statements and use cases help prioritisation.

---

## Stay Informed

- Follow updates via the GitHub repositories  
- Connect on LinkedIn  
- Review Steering Committee summaries
